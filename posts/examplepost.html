첫 번째 글: GitHub 블로그 시작하기

2025-11-10 · 카테고리: 개발

안녕하세요! 이 글은 GitHub Pages로 만든 블로그의 첫 번째 포스트입니다.

앞으로 이곳에 개발 공부 내용과 개인적인 생각들을 정리해보려고 합니다.

1. 저장소 만들기

GitHub에서 새 저장소를 만들고, index.html 파일을 추가합니다.

2. GitHub Pages 켜기

Settings > Pages로 들어가서 브랜치를 선택하면 블로그가 배포됩니다.

3. 앞으로 할 일

  • 레이아웃 조금 더 다듬기
  • 글 목록 자동화(Jekyll 사용해보기)
  • 다크 모드 추가해보기
